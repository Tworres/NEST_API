## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

This is a sample with already integrated many of resources that is usually used in a API

## Installation

```bash
$ npm install
```

## Running the Docker

```bash
# development
$ npm run db:dev:up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
``
